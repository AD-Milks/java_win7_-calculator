##### 介绍

- 仿造 Win 7 计算器，实现基本计算器的功能；

- 与标准计算器相比较大大的降低了数字计算的难度，并且提高了计算的准确度和精确度

- 支持复制 ，粘贴快捷键

##### 安装教程

1. 把本程序代码导入 Eclipse 或 IDEA，运行**main.java**文件即可！
2. **由于代码用到了 JDK1.6 的流式编程语法，因此至少要 JDK1.6 才能运行本程序**
